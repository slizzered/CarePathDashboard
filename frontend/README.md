Carepath Dashboard - frontend
============================================================


Introduction
------------

This is the User interface for the CarePathDashboard (from CarLu Solutions).

### How to get started

```
# open frontend in a IDE (e.g. VSC)

# install dependencies
npm install

# run web server
npm run dev

# open browser at 'http://localhost:3000'
```
